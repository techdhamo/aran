## Compliance Mapping

### OWASP MASVS (Mobile App Security Verification Standard)

Aran natively fulfills **MASVS-RESILIENCE (L2 + R)**:

- **MSTG-RESILIENCE-1:** Root/Jailbreak detection (Magisk/Zygisk).
- **MSTG-RESILIENCE-2:** Anti-debugging (`ptrace`).
- **MSTG-RESILIENCE-4:** Dynamic instrumentation detection (Frida).
- **MSTG-RESILIENCE-9:** Obfuscation (O-LLVM).

### RBI Digital Payment Security Guidelines

Aran empowers Indian banks to meet strict regulatory mandates:

- **Section 3.2 (Device Binding):** Aran hardware fingerprinting ensures strict app-to-device binding.
- **Section 4.1 (Malware/Overlay Protection):** "Screen Shield" blocks overlay attacks from banking trojans.
- **Section 6.3 (Fraud Risk Management):** Continuous session monitoring via eFRM fulfills RBI’s risk management mandate.
