## The Duality Brand Pitch

# 

"Modern threats don't attack the firewall or the endpoint; they attack the blind spots in between. **Mazhai** (The Rain) provides fluid cloud intelligence, while **Aran** (The Fortress) provides tactical edge defense. We sell an intelligent ecosystem that renders siloed tools obsolete."

## Competitive Matrix (Battlecards)

# 

*   **Aran vs. Appdome:** Aran uses a lightweight C++ JNI core with zero CI/CD bloat and deep backend AI correlation.
    
*   **Aran vs. Imperva:** Aran WAAP shares context with mobile sensors to stop logic attacks (BOLA/IDOR) that regex-based WAFs miss.
    
*   **Aran vs. Datadog:** Aran ASPM correlates static vulnerabilities with _live_ RASP attacks to eliminate developer alert fatigue.
    

## Compliance Mapping

# 

*   **MSTG-RESILIENCE-1:** Root/Jailbreak detection (Magisk/Zygisk).
    
*   **MSTG-RESILIENCE-2:** Anti-debugging (`ptrace`).
    
*   **MSTG-RESILIENCE-4:** Hooking detection (Frida).
    
*   **MSTG-RESILIENCE-9:** Obfuscation (O-LLVM).
    

### RBI Digital Payment Security Guidelines

# 

*   **Section 3.2 (Device Binding):** Hardware fingerprinting ensures app-to-device binding.
    
*   **Section 4.1 (Malware/Overlay Protection):** "Screen Shield" blocks overlay attacks from trojans.
    
*   **Section 6.3 (Fraud Risk Management):** Continuous behavioral monitoring via eFRM.
